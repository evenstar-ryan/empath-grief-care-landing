# Empath Grief Care Support Groups Website

A single-page website displaying grief support group schedules and information for Empath Health across multiple Florida service areas.

## Overview

This website provides information about grief support groups offered by Empath Health, including:
- Marion County (Hospice of Marion County Area)
- Hillsborough & Pinellas Counties (Suncoast Hospice Areas)
- DeSoto, Charlotte, Manatee, and Sarasota Counties (Tidewell Hospice Area)
- Broward & Palm Beach Counties (Trustbridge Area)

## Features

- Responsive design that works on desktop, tablet, and mobile
- Sticky filter bar for quick navigation between service areas
- Interactive group sessions with hover effects
- Click-to-call phone numbers for mobile users
- Virtual event tags and special event highlighting
- Empath Health branded color scheme (purple and teal)

## Deployment
